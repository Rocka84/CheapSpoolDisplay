CheapSpoolDisplay Installer

Welcome! Connect your ESP32 Cheap Yellow Display (CYD) via USB to flash the latest firmware directly from your browser, securely.

Note: Supported on Google Chrome and Microsoft Edge on Desktop OS.

Configuration Instructions

After the flash is complete or if your device is already flashed, you can use the Serial Console below to configure Wi-Fi and Webhooks over USB!

  1. Click Connect Console below and choose your CYD's COM port.
  2. Set Wi-Fi: Type set ssid YourNetworkName and hit enter.
  3. Set Password: Type set pass YourPassword and hit enter.
  4. Set Snapmaker Webhook: Type set url http://your-server-ip:port/webhook
  5. Verify your settings with: get config
